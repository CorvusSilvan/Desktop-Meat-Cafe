## Monk's Cafe Management System

Консольна програма для керування замовленнями в кафе.

### Основний функціонал:
- Перегляд об'єднаного меню (4 м'ясні страви + 4 напої)
- Додавання та видалення позицій з замовлення
- Додавання чайових (у відсотках або фіксованій сумі)
- Перегляд повного рахунку з деталізацією (Net Total, Tip, GST, Total Amount)
- Очищення замовлення
- Збереження та завантаження замовлень у файл (формат: Menu-{номер}.txt)
- Керування власним номером замовлення

### Як використовувати:
1. Запустіть програму та введіть номер замовлення (цифри, до 10 символів)
2. Оберіть дію з головного меню:
   - `1. Add Item` - додати позицію
   - `2. Remove Item` - видалити позицію
   - `3. Add Tip` - додати чайові
   - `4. Display Bill` - показати рахунок
   - `5. Clear All` - очистити замовлення
   - `6. Save to file` - зберегти у файл
   - `7. Load from file` - завантажити з файлу
   - `8. Exit` - вихід


### Особливості:
- При додаванні існуючої позиції збільшується її кількість
- При видаленні: якщо кількість > 1 - зменшується на 1, якщо = 1 - видаляється повністю
- Чайові додаються як окрема позиція у замовленні
- Файли зберігаються у форматі: `Назва|Кількість|Ціна`

### Вимоги:
